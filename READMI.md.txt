# 📊 Customer Segmentation Dashboard

This project presents a Power BI dashboard for analyzing customer behavior based on income and spending score.

## 🔹 Features
- KPI metrics (Total Customers, Avg Income, Avg Spending)
- Customer Segmentation
- Interactive Visualizations

## 📈 Insights
- Premium customers have higher income and spending
- Clear segmentation helps in targeted marketing

## 🛠 Tools Used
- Power BI
- Data Analysis
